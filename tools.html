  • Extract RSS feed from YouTube Channel

    It's not limited to YouTube links (also Invidious and every other site that contains a <link> ... type=application/rss+xml</link> html tag. Returns first matching result

    Usage: Provide channel uri as parameter (e.g. https://tools.jaott.de/rss/?url=https://www.youtube.com/...)
